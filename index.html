What Delimit Does

Delimit compares two versions of an OpenAPI specification and detects 27 types of changes — 17 breaking and 10 non-breaking. It classifies changes by severity, recommends semver bumps, generates migration guides, and enforces custom policies.

How It Works

  1. Diff — Compare old and new OpenAPI specs to find all changes.
  2. Classify — Each change is categorized as breaking or non-breaking with a severity level.
  3. Enforce — Apply policy presets or custom YAML rules to pass/fail the change.
  4. Report — Generate PR comments, CI annotations, and migration guides.
