MCP Server

Use Delimit with AI coding assistants via the Model Context Protocol (MCP).

Delimit provides an MCP server that exposes API governance tools to any MCP-compatible AI assistant, including Claude Code, OpenAI Codex CLI, and Google Gemini CLI.

Quick Setup

npx delimit-cli mcp

Available MCP Tools

The Delimit MCP server exposes the following core tools:

ToolDescription
delimit_lintLint two OpenAPI specs for breaking changes and policy violations.
delimit_diffPure diff between two specs. Lists all changes.
delimit_test_generateGenerate test skeletons from source code.
delimit_test_coverageAnalyze test coverage.

Integration Guides

  • Claude Code — Full MCP setup for Claude Code
  • OpenAI Codex CLI — MCP setup for Codex
  • Google Gemini CLI — MCP setup for Gemini

Manual Configuration

Add to your .mcp.json or equivalent config:

{
  "mcpServers": {
    "delimit": {
      "command": "npx",
      "args": ["-y", "delimit-cli@latest", "mcp"]
    }
  }
}

Usage Examples

Once connected, ask your AI assistant:

  • "Lint my OpenAPI spec for breaking changes"
  • "Compare my old and new API specs"
  • "What breaking changes are in this PR?"
  • "Generate tests for my API endpoints"
