CLI Reference

All Delimit CLI commands and options.

Installation

# Run without installing
npx delimit-cli [command]

# Install globally
npm install -g delimit-cli

# Verify installation
delimit --version

Commands

delimit init [--preset]

Initialize Delimit in your project. Creates a .delimit/ directory with default configuration.

# Use default preset
delimit init

# Use strict preset (all violations are errors)
delimit init --preset strict

# Use relaxed preset (all violations are warnings)
delimit init --preset relaxed

delimit lint

Lint two OpenAPI specs for breaking changes and policy violations. This is the primary CI command.

delimit lint --old api/v1.yaml --new api/v2.yaml
delimit lint --old api/v1.yaml --new api/v2.yaml --policy .delimit/policies.yml

delimit diff

Pure diff between two specs. Lists all changes without applying policy rules.

delimit diff --old api/v1.yaml --new api/v2.yaml

delimit explain

Explain a specific change type or policy rule in detail.

delimit explain endpoint_removed
delimit explain no_type_changes

delimit doctor

Diagnose your Delimit setup. Checks configuration, spec files, and policy validity.

delimit doctor

delimit setup

Interactive setup wizard. Configures Delimit for your project, installs hooks, and validates your spec.

delimit setup

delimit activate

Activate the governance layer. Installs pre-commit hooks and PATH shims for continuous governance.

delimit activate
