Claude Code Integration

Set up Delimit API governance in Claude Code.

Prerequisites

  • Claude Code installed and configured
  • Node.js 18+ installed

Configuration

# Add to ~/.mcp.json or .mcp.json in your project
{
  "mcpServers": {
    "delimit": {
      "command": "npx",
      "args": ["-y", "delimit-cli@latest", "mcp"]
    }
  }
}

Usage

Once configured, you can ask Claude to:

  • "Lint my OpenAPI spec for breaking changes"
  • "Compare the old and new API specs in this directory"
  • "What breaking changes are in my current changes?"
  • "Generate tests for my source files"

Claude Code Tips

Claude Code automatically discovers MCP servers from .mcp.json in your project root or ~/.mcp.json for global access. The Delimit tools will appear alongside Claude's built-in tools.

Available Tools

The Delimit MCP server provides these tools to Claude:

ToolDescription
delimit_lintLint two OpenAPI specs for breaking changes and policy violations
delimit_diffDiff two OpenAPI specs and list all changes
delimit_test_generateGenerate test skeletons for source code
delimit_test_coverageAnalyze test coverage

Troubleshooting

  • Ensure npx is available in your PATH
  • Check that the MCP server starts without errors: npx delimit-cli mcp
  • Verify your OpenAPI spec files are valid YAML or JSON
